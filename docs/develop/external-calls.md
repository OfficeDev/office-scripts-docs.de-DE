## <a name="retrieve-information-with-fetch"></a><span data-ttu-id="aa097-130">Abrufen von Informationen mit `fetch`</span><span class="sxs-lookup"><span data-stu-id="aa097-130">Retrieve information with `fetch`</span></span>

<span data-ttu-id="aa097-131">Die [Abruf-API](https://developer.mozilla.org/docs/Web/API/Fetch_API) ruft Informationen von externen Diensten ab.</span><span class="sxs-lookup"><span data-stu-id="aa097-131">The [fetch API](https://developer.mozilla.org/docs/Web/API/Fetch_API) retrieves information from external services.</span></span> <span data-ttu-id="aa097-132">Es handelt sich um eine `async` API, daher müssen Sie die `main` Signatur Ihres Skripts anpassen.</span><span class="sxs-lookup"><span data-stu-id="aa097-132">It is an `async` API, so you need to adjust the `main` signature of your script.</span></span> <span data-ttu-id="aa097-133">Machen Sie die `main` `async` Funktion, und lassen Sie sie eine `Promise<void>` zurückgeben.</span><span class="sxs-lookup"><span data-stu-id="aa097-133">Make the `main` function `async` and have it return a `Promise<void>`.</span></span> <span data-ttu-id="aa097-134">Sie sollten auch sicher sein, `await` den `fetch` Anruf und `json` abzurufen.</span><span class="sxs-lookup"><span data-stu-id="aa097-134">You should also be sure to `await` the `fetch` call and `json` retrieval.</span></span> <span data-ttu-id="aa097-135">Dadurch wird sichergestellt, dass diese Vorgänge abgeschlossen sind, bevor das Skript beendet wird.</span><span class="sxs-lookup"><span data-stu-id="aa097-135">This ensures those operations complete before the script ends.</span></span>

<span data-ttu-id="aa097-136">Alle JSON-Daten, die von abgerufen `fetch` werden, müssen mit einer im Skript definierten Schnittstelle übereinstimmen.</span><span class="sxs-lookup"><span data-stu-id="aa097-136">Any JSON data retrieved by `fetch` must match an interface defined in the script.</span></span> <span data-ttu-id="aa097-137">Der zurückgegebene Wert muss einem bestimmten Typ zugewiesen werden, da [Office Skripts den `any` Typ nicht unterstützen.](typescript-restrictions.md#no-any-type-in-office-scripts)</span><span class="sxs-lookup"><span data-stu-id="aa097-137">The returned value must be assigned to a specific type because [Office Scripts do not support the `any` type](typescript-restrictions.md#no-any-type-in-office-scripts).</span></span> <span data-ttu-id="aa097-138">Sie sollten in der Dokumentation für Ihren Dienst nach den Namen und Typen der zurückgegebenen Eigenschaften lesen.</span><span class="sxs-lookup"><span data-stu-id="aa097-138">You should refer to the documentation for your service to see what the names and types of the returned properties are.</span></span> <span data-ttu-id="aa097-139">Fügen Sie dann die entsprechende Schnittstelle oder Schnittstellen zu Ihrem Skript hinzu.</span><span class="sxs-lookup"><span data-stu-id="aa097-139">Then, add the matching interface or interfaces to your script.</span></span>

<span data-ttu-id="aa097-140">Das folgende Skript `fetch` verwendet, um JSON-Daten vom Testserver in der angegebenen URL abzurufen.</span><span class="sxs-lookup"><span data-stu-id="aa097-140">The following script uses `fetch` to retrieve JSON data from the test server in the given URL.</span></span> <span data-ttu-id="aa097-141">Beachten Sie die `JSONData` Schnittstelle, um die Daten als übereinstimmenden Typ zu speichern.</span><span class="sxs-lookup"><span data-stu-id="aa097-141">Note the `JSONData` interface to store the data as a matching type.</span></span>

```TypeScript
async function main(workbook: ExcelScript.Workbook): Promise<void> {
  // Retrieve sample JSON data from a test server.
  let fetchResult = await fetch('https://jsonplaceholder.typicode.com/todos/1');

  // Convert the returned data to the expected JSON structure.
  let json : JSONData = await fetchResult.json();

  // Display the content in a readable format.
  console.log(JSON.stringify(json));
}

/**
 * An interface that matches the returned JSON structure.
 * The property names match exactly.
 */
interface JSONData {
  userId: number;
  id: number;
  title: string;
  completed: boolean;
}
```
